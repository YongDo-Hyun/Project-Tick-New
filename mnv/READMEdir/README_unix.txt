README_unix.txt for version 10.0 of MNV: MNV is not Vim.

This file explains the installation of MNV on Unix systems.
See "README.txt" for general information about MNV.


When you use the source distribution, "make install" is used to install MNV.
See the "INSTALL" file in the "src" directory.

If you use a compiled package, follow the instructions for the package.
